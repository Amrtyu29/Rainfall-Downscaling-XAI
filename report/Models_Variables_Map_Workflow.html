1 The variables — 7 physical predictors + 4 helpers (same everywhere)

q850Specific humidity @850 hPa — moisture amount
rh850Relative humidity @850 hPa — nearness to saturation
u850Zonal (E–W) wind @850 hPa — moisture transport
v850Meridional (N–S) wind @850 hPa — transport
z500Geopotential height @500 hPa — mid-level circulation
mslpMean sea-level pressure — weather systems
t2m2-metre temperature — thermodynamic state
lat, lonLocation — where the grid cell is
month_sinCyclic month encoding — when (season)
month_cosCyclic month encoding — when (season)
FEATURES = 7 predictors + lat + lon + month_sin + month_cos = 11 inputs  →  target = IMD rainfall  (config.py)

All ML models share the same 7-predictor physical basis — only the data changes (ERA5 monthly → GCM future → NCEP daily). The ETCCDI indices use no predictors.
